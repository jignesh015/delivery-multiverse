***
>NOTE: This project was primarily created for Brackeys Game Jam 2026.1, built within the jam timeframe (7 days) as a fast-paced experimental concept.
***

FedUp Ex is a chaotic delivery game where one small mistake spirals into interdimensional disaster. You play as a delivery driver who accidentally drops a highly sensitive package, tearing open a portal to strange and unstable dimensions. Now every delivery run risks warping reality itself.

Drive, survive, and try to keep your job while the universe glitches around you.



***

## 🛠 Prerequisites & Licensing
Important: This project utilizes the Feel library by More Mountains for game juice and juice effects.

Due to the Unity Asset Store EULA, the source files for the Feel library are not included in this repository. To run or develop this project, you must:
 - Purchase a license for Feel on the Unity Asset Store.
 - Import the package into the project via the Unity Package Manager.
 - Ensure the files are located in the Assets/Feel/ directory.

Note: If you do not have Feel installed, the project will still open, but you will encounter compiler errors and missing script references (MMFeedbacks) in most scenes.
